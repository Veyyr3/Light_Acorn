Light Acorn has several git branches.

Main branches:
* master -- main stable branch.
* dev -- developming branch.

Acorn Architecture Developming:
* acorn -- only ACORN-way branch. Code from `acorn` branch put into `dev`.
* reacorn -- only REACORN-way branch. Code from `reacorn` branch put into `dev`.

Acorn Tools Developming:
* game -- only for game tools.

Acorn Experimental:
* experimental -- for tests branch.
* game_experimental -- only for game experiments. Code from `game_experimental` put into `game` branch.
